The basic <kbd>.nav</kbd> pattern class throws elements inline, sets the
project-defined <kbd>$font-weight</kbd> and <kbd>$font-family</kbd> values.

<ul class="nav">
  <li><a href="#">Home</a></li>
  <li><a href="#">Menu</a></li>
  <li class="active">Drinks</li>
  <li><a href="#">Meat</a></li>
</ul>
